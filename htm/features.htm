Features

CoPilot RI provides the following features:

  • Hover in place
  • Pilot to a mouse click
  • Pilot to keyed in coordinates
  • Pilot to a series of coordinates (Survey)
  • Pilot on a heading at a constant speed and depth (or altitude) (Cruise Control)
  • Transparent manual operation (take over manual control at any time)
  • Navigation display
  • GPS integration
  • Sonar overlay
  • Chart overlay
  • Markers
  • Recording
  • Offline planning of missions

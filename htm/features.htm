Features

CoPilot RI provides the following features:

  • Transparent manual operation
  • Hover in place
  • Pilot to a mouse click
  • Pilot to keyed in coordinates
  • Pilot to a series of coordinates (Survey)
  • Pilot on a heading at a constant speed (Cruise Control)
  • Navigation display
  • Sonar overlay
  • GPS integration
  • Charts overlay
  • Markers
  • Recording
  • Offline planning of missions
